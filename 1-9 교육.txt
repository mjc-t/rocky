인터넷 이용 시
1. /etc/hosts		호스트 파일
2. /etc/resolv.conf		DNS 서비스

DNS(Domain Name System): 도메인 이름 체계
최상위: .
Top 레벨 도메인 - gTLD, ccTLD
	gTLD: .com, .net, .org, .int, .edu, .mil, .gov
	ccTLD: .kr, .jp, .cn, .tw, .de, .dk, .at, .io, .to, .nu, .tv
Second 레벨 도메인
	samsung.com
	.co.kr	.go.kr	.hs.kr	ms.kr	es.kr	or.kr	ne.kr	...

캐싱 전용 서버
1. dnf install bind bind-chroot
2. vi /etc/named.conf
3. systemctl {start | enable | status} named
4. firewall-cmd --permanent --add-service=dns
   firewall-cmd --reload
   firewall-cmd --list-services

127.0.0.1: Loopback 주소
nslookup, dig: 호스트네임 -> IP 주소로 확인

[트러블슈팅]
1. ipconfig			ip addr, nmcli dev show
2. ping 127.0.0.1
3. ping 자기주소
4. ping 게이트웨이주소
5. ping 원격주소		브라우저: google.com

텍스트 브라우저: elinks, curl, lynx
