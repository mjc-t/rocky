클라이언트: 테스트

메일 서버 전제조건
- 호스트 네임 변경: hostnamectl set-hostname FQDN

메일서버 구현시 필수 패키지: sendmail, sendmail-cf, dovecot
1. dnf -y install sendmail sendmail-cf dovecot
2. 환경설정
/etc/mail/sendmail.cf
/etc/mail/access
makemap hash /etc/mail/access < /etc/mail/access
/etc/dovecot/dovecot.conf
/etc/dovecot/conf.d/10-ssl.conf
/etc/dovecot/conf.d/10-mail.conf
3. systemctl restart sendmail
   systemctl enable sendmail
   systemctl restart dovecot
   systemctl enable dovecot
4. firewall-cmd --permanent --add-service=smtp
   firewall-cmd --permanent --add-service=pop3
   firewall-cmd --permanent --add-service=imap
   firewall-cmd --reload
   firewall-cmd --list-services

NFS:		파일 공유-리눅스 사이
SAMBA:	      //    -리눅스와 윈도우 사이

1. dnf -y install nfs-utils
2. vi /etc/exports
	/share  *(rw,sync)
   mkdir /share
   chmod 707 /share
3. systemctl restart nfs-server
   systemctl enable nfs-server
   exportfs -v 	공유여부 확인
4. firewall-cmd --permanent --add-service=nfs
   firewall-cmd --permanent --add-service=mountd
   firewall-cmd --permanent --add-service=rpc-bind
   firewall-cmd --reload
   firewall-cmd --list-services
=====
nfs 클라이언트
showmount -e 192.168.111.100
mkdir myShare
su -c 'mount -t nfs 192.168.111.100:/share myShare'
